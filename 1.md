## 流程记录

#### 关联远程仓库
````
  git init -y //初始化仓库 

  git remote add [仓库地址]
````
###### 或者利用clone关联
````
  git clone [仓库地址]
````
#### 查看或者新建分支
````
  git branch //查看分支
  git branch [分支名] //新建分支
  git branch -d [分支名] //删除分支
  git branch -m [分支名] [新的分支名] //修改分支名

  git checkout -b [分支名] //新建并切换
````
#### 查看相关命令
````
  git status //查看当前版本的文件状态
  
  git log  //查看当前版本的历史提交
  git log --pretty==oneline
  git reflog //查看所有历史记录
````
#### 回滚
````
  git reset --hard [版本号]
````
### 文件日常使用较多的命令
````
  git pull origin master //拉取
  git add .
  git commit -m''
  git push origin master //推送
````
### 分支合并
````
  git merge [目标分支]
````
